Lab02 - HTML Cơ bản

Chào mừng bạn đến với bài thực hành HTML cơ bản. Dưới đây là danh sách các bài tập:

Danh sách bài tập

  • Bài 1: Trang giới thiệu bản thân (about.html)
    Thực hành các thẻ cơ bản: heading, paragraph, list, image, link
  • Bài 2: Bảng lịch học (schedule.html)
    Thực hành tạo bảng với table, tr, th, td
  • Bài 3: Form liên hệ (contact.html)
    Thực hành tạo form với input, select, textarea, button
  • Bài 4: Trang web với Semantic HTML (semantic.html)
    Thực hành sử dụng các thẻ semantic: header, nav, main, section, article, aside, footer
  • Bài 5: Thực hành tổng hợp (practice.html)
    Kết hợp tất cả kiến thức đã học

Mục tiêu học tập

Sau khi hoàn thành Lab02, bạn sẽ:

  • Hiểu được cấu trúc cơ bản của tài liệu HTML
  • Sử dụng thành thạo các thẻ HTML cơ bản
  • Tạo được bảng và form HTML
  • Áp dụng được các thẻ semantic HTML
  • Xây dựng được trang web đơn giản nhưng có cấu trúc rõ ràng

Ghi chú

Lưu ý: Hãy mở từng file HTML bằng trình duyệt để xem kết quả. Bạn có thể sử dụng Live Server extension trong VS Code để xem preview trực tiếp.
