🧮 Arithmetic Operators

Arithmetic operators perform mathematical calculations on numbers:

Basic Math Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (Remainder)
  • ** Exponentiation

Assignment Operators

  • = Assignment
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign
  • %= Modulus and assign

Increment/Decrement

  • ++ Increment by 1
  • -- Decrement by 1
  • x++ Post-increment
  • ++x Pre-increment
  • x-- Post-decrement
  • --x Pre-decrement
// Arithmetic operators examples
let a = 10;
let b = 3;

console.log(a + b);    // 13 - Addition
console.log(a - b);    // 7 - Subtraction
console.log(a * b);    // 30 - Multiplication
console.log(a / b);    // 3.333... - Division
console.log(a % b);    // 1 - Modulus (remainder)
console.log(a ** b);   // 1000 - Exponentiation

// Assignment operators
let x = 5;
x += 3;               // x = x + 3 = 8
x *= 2;               // x = x * 2 = 16

// Increment/Decrement
let count = 0;
console.log(count++);   // 0 (post-increment: use then increment)
console.log(++count);   // 2 (pre-increment: increment then use)

⚖️ Comparison Operators

Comparison operators compare values và return boolean results (true or false):

Equality Operators

  • == Equal (loose)
  • === Equal (strict)
  • != Not equal (loose)
  • !== Not equal (strict)

Relational Operators

  • < Less than
  • > Greater than
  • <= Less than or equal
  • >= Greater than or equal
// Equality comparisons
console.log(5 == "5");     // true (loose equality - type coercion)
console.log(5 === "5");    // false (strict equality - no type coercion)
console.log(5 != "5");     // false
console.log(5 !== "5");    // true

// Relational comparisons
console.log(10 > 5);      // true
console.log(10 < 5);      // false
console.log(10 >= 10);    // true
console.log(10 <= 5);     // false

// String comparison (alphabetical)
console.log("apple" < "banana");  // true
console.log("A" < "a");            // true (ASCII comparison)

🧠 Logical Operators

Logical operators perform logical operations và are commonly used with boolean values:

Logical Operators

  • && AND (both true)
  • || OR (at least one true)
  • ! NOT (opposite)

Truth Tables

AND (&&) Operator

A B A && B
true true true
true false false
false true false
false false false

OR (||) Operator

A B A || B
true true true
true false true
false true true
false false false 
// Logical operators examples
let age = 25;
let hasLicense = true;
let hasInsurance = false;

// AND operator (&&) - all conditions must be true
let canDrive = age >= 18 && hasLicense;
console.log(canDrive);          // true

// OR operator (||) - at least one condition must be true
let needsDocuments = !hasLicense || !hasInsurance;
console.log(needsDocuments);   // true

// NOT operator (!) - reverses boolean value
console.log(!true);          // false
console.log(!false);         // true
console.log(!hasInsurance);   // true

// Short-circuit evaluation
console.log(true || "This won't be evaluated");  // true
console.log(false && "This won't be evaluated"); // false

📊 Operator Precedence

JavaScript follows specific rules về thứ tự operators are evaluated. Higher precedence operators are evaluated first:

Precedence Levels (Highest to Lowest)

1
Parentheses: ( )
Grouping và function calls
2
Member Access: . []
Object property access
3
Unary: ! - + ++ --
Logical NOT, unary minus/plus, increment/decrement
4
Exponentiation: **
Power operation (right-to-left)
5
Multiplication/Division: * / %
Arithmetic operations
6
Addition/Subtraction: + -
Arithmetic operations
7
Relational: < <= > >=
Comparison operations
8
Equality: == != === !==
Equality comparisons
9
Logical AND: &&
Logical conjunction
10
Logical OR: ||
Logical disjunction
11
Assignment: = += -= *= /= %=
Variable assignments (right-to-left) 
// Operator precedence examples
console.log(2 + 3 * 4);        // 14 (not 20) - multiplication first
console.log((2 + 3) * 4);      // 20 - parentheses override precedence

console.log(5 > 3 && 2 < 4);   // true - comparison before logical AND
console.log(true || false && false); // true - AND before OR
console.log((true || false) && false); // false - parentheses change order

// Assignment is right-to-left
let a, b, c;
a = b = c = 5;           // All variables get value 5
console.log(a, b, c);     // 5 5 5
