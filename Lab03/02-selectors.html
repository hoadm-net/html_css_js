Bài 2: CSS Selectors (Bộ Chọn CSS)

CSS Selectors giúp chúng ta chọn và styling các element HTML. Hãy cùng tìm hiểu các loại selector cơ bản!

TAG SELECTOR

1. Tag Selector

Chọn tất cả element cùng loại tag. Ví dụ: h1, p, div

h1 { color: white; text-align: center; } p { line-height: 1.6; margin: 15px 0; }

Tất cả thẻ <p> trên trang này đều được style bằng tag selector!

CLASS SELECTOR

2. Class Selector

Chọn element có class cụ thể. Sử dụng dấu chấm (.) trước tên class.

.highlight { background: linear-gradient(45deg, #ff6b6b, #4ecdc4); color: white; padding: 15px; border-radius: 8px; }
Element này sử dụng class="highlight"
Lưu ý: Class có thể sử dụng cho nhiều element!
Ưu điểm: Tái sử dụng được, dễ bảo trì!
ID SELECTOR

3. ID Selector

Chọn element có ID cụ thể. Sử dụng dấu thăng (#) trước tên ID.

#header-special { background: linear-gradient(90deg, #ff7e5f, #feb47b); color: white; padding: 20px; }

Element này có id="header-special"

ID chỉ nên sử dụng cho một element duy nhất!

DESCENDANT SELECTOR

4. Descendant Selector

Chọn element con bên trong element cha. Sử dụng dấu cách.

.menu-demo ul li { background: white; padding: 12px; border-left: 4px solid #007bff; } .menu-demo ul li a { color: #007bff; text-decoration: none; }

CHILD SELECTOR

5. Child Selector (Direct Child)

Chọn element con trực tiếp. Sử dụng dấu >

.parent-demo > p { background: #ffeaa7; padding: 10px; border-left: 4px solid #fdcb6e; }

Paragraph này là con trực tiếp của .parent-demo

Paragraph này không được style vì nó là con của div, không phải con trực tiếp của .parent-demo

Paragraph này lại được style vì là con trực tiếp

MULTIPLE CLASSES

6. Multiple Classes

Một element có thể có nhiều class, và ta có thể target chính xác bằng cách kết hợp các class.

.box.red { background: #ff6b6b; } .box.blue { background: #4ecdc4; } .box.green { background: #95e1a3; }
Box với class="box red"
Box với class="box blue"
Box với class="box green"
PSEUDO-CLASSES

7. Pseudo-classes

Chọn element dựa trên trạng thái đặc biệt như :hover, :active, :focus

.button-demo:hover { background: #0056b3; transform: scale(1.05); } .button-demo:active { transform: scale(0.95); }
Hover me! Click me! Try me!
CSS SPECIFICITY

8. CSS Specificity (Độ Ưu Tiên)

Khi có nhiều rule CSS cùng target một element, thứ tự ưu tiên sẽ là:

  1. Inline styles (style="...") - 1000 điểm
  2. IDs (#myId) - 100 điểm
  3. Classes, pseudo-classes, attributes (.myClass, :hover) - 10 điểm
  4. Elements (div, p, a) - 1 điểm
Lưu ý: Nếu điểm bằng nhau, CSS rule nào viết sau sẽ được áp dụng!

🎯 Thực hành

Hãy mở Developer Tools (F12) và inspect các element để xem CSS selectors hoạt động!
