Flexbox Basics

Tạo layout linh hoạt và responsive với Flexbox

1. Giới Thiệu Flexbox

Flexbox (Flexible Box Layout) là một phương pháp layout 1 chiều giúp:

  • Căn chỉnh items dễ dàng
  • Phân phối space linh hoạt
  • Tạo responsive layout
  • Giải quyết các vấn đề layout truyền thống
/* Tạo flex container */ .container { display: flex; } /* Flex items tự động trở thành flexible */ .item { flex: 1; /* Chia đều space */ }
2. Flex Direction
flex-direction: row; /* Mặc định - ngang từ trái sang phải */ flex-direction: row-reverse; /* Ngang từ phải sang trái */ flex-direction: column; /* Dọc từ trên xuống dưới */ flex-direction: column-reverse; /* Dọc từ dưới lên trên */

10. Patterns Thường Dùng
/* Perfect centering */ .center { display: flex; justify-content: center; align-items: center; height: 100vh; } /* Sidebar layout */ .layout { display: flex; } .sidebar { flex: 0 0 250px; } .content { flex: 1; } /* Equal height columns */ .columns { display: flex; } .column { flex: 1; padding: 20px; } /* Sticky footer */ .page { display: flex; flex-direction: column; min-height: 100vh; } .content { flex: 1; } .footer { flex: 0 0 auto; }

🎯 Tips cho Flexbox:

  • Flexbox hoạt động trên 2 axis: main axis và cross axis
  • justify-content điều khiển main axis, align-items điều khiển cross axis
  • Sử dụng flex: 1 để items chia đều space
  • Kết hợp flex-wrap với min-width để tạo responsive grid
  • Flexbox tốt cho 1D layouts, dùng Grid cho 2D layouts
