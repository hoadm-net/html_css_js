8. Responsive với Display
Resize browser để xem thay đổi!
/* Mobile First Approach */ .element { display: block; } @media (min-width: 768px) { .element { display: inline-block; } } /* Hide on mobile */ @media (max-width: 767px) { .desktop-only { display: none; } }

🎯 Tips cho Display & Position:

  • Sử dụng Flexbox/Grid thay vì float cho layout hiện đại
  • Position absolute cần parent có position relative
  • Fixed position luôn relative to viewport
  • Z-index chỉ hoạt động với positioned elements
  • Sticky position cần thiết lập top/bottom/left/right
