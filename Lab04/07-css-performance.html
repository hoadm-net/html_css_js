1 CSS Performance Fundamentals

Hiểu rõ browser rendering pipeline và cách CSS ảnh hưởng đến performance.

/* ✅ Efficient CSS - Affects only composite layer */ .good-animation { transition: transform 0.3s ease, opacity 0.3s ease; } .good-animation:hover { transform: translateY(-5px) scale(1.02); opacity: 0.9; } /* ❌ Expensive CSS - Triggers layout recalculation */ .bad-animation { transition: width 0.3s ease, height 0.3s ease, margin 0.3s ease; } .bad-animation:hover { width: 220px; /* Forces layout */ height: 110px; /* Forces layout */ margin-left: 10px; /* Forces layout */ }

2 Critical CSS & Loading Strategies

Tối ưu loading với critical CSS, async loading và resource hints.

<!-- Critical CSS - Inline trong <head> --> <style> /* Above-the-fold styles */ .header, .hero, .nav { /* Critical styles here */ } </style> <!-- Non-critical CSS - Load async --> <link rel="preload" href="non-critical.css" as="style" onload="this.onload=null;this.rel='stylesheet'"> <!-- Resource hints for performance --> <link rel="dns-prefetch" href="//fonts.googleapis.com"> <link rel="preconnect" href="//fonts.gstatic.com" crossorigin> <link rel="prefetch" href="next-page.css">

3 CSS Containment & Isolation

Sử dụng CSS containment để isolate performance impacts.

/* CSS Containment - Isolate rendering costs */ .component { contain: layout style paint; /* contain: strict; - Applies all containment */ } /* Size containment */ .fixed-size { contain: size; width: 300px; height: 200px; } /* Layout containment */ .isolated-layout { contain: layout; /* Layout changes won't affect parent */ } /* Paint containment */ .paint-boundary { contain: paint; /* Ensures paint doesn't overflow */ }

4 Performance-Optimized Animations

Tạo animations mượt mà 60fps với GPU acceleration.

/* ✅ GPU-accelerated animations */ .smooth-element { will-change: transform; transition: transform 0.3s ease; } .smooth-element:hover { transform: translateZ(0) scale(1.1); /* Force GPU layer */ } /* ✅ Efficient keyframes */ @keyframes optimized-bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-20px); } } /* ❌ Avoid animating these properties */ .avoid { /* width, height, margin, padding, border */ /* top, left, right, bottom */ /* font-size (use transform: scale instead) */ } /* 🚀 Use these instead */ .prefer { transform: translateX() translateY() scale() rotate(); opacity: 0-1; filter: blur() brightness() contrast(); }

5 Performance Metrics & Monitoring

Đo lường và monitor CSS performance với Web Vitals.

/* Performance monitoring với CSS */ .monitor-layout { /* Use CSS custom properties for dynamic values */ --layout-shift: 0; --paint-time: 0; } /* Optimize for Core Web Vitals */ .lcp-optimized { /* Largest Contentful Paint optimization */ content-visibility: auto; contain-intrinsic-size: 300px; } .cls-safe { /* Cumulative Layout Shift prevention */ width: 100%; height: auto; aspect-ratio: 16/9; } /* First Input Delay optimization */ .fid-optimized { /* Reduce JavaScript blocking */ will-change: transform; pointer-events: none; }

6 Responsive Performance Optimization

Tối ưu responsive design cho multiple devices và connection speeds.

/* Mobile-first performance optimization */ .responsive-grid { display: grid; grid-template-columns: 1fr; gap: 1rem; } /* Efficient breakpoints */ @media (min-width: 640px) { .responsive-grid { grid-template-columns: repeat(2, 1fr); } } @media (min-width: 1024px) { .responsive-grid { grid-template-columns: repeat(3, 1fr); } } /* Connection-aware loading */ @media (prefers-reduced-motion: reduce) { * { animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; transition-duration: 0.01ms !important; } } /* Dark mode performance */ @media (prefers-color-scheme: dark) { :root { --bg: #1a1a1a; --text: #ffffff; } }
