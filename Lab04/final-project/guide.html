Thiết lập môi trường và tải template cần thiết

📁 Tạo thư mục dự án

Tạo thư mục mới cho portfolio của bạn và mở trong VS Code:

mkdir my-portfolio cd my-portfolio code .

📄 Tải template

Sao chép nội dung từ template.html vào file index.html mới:

💡 Mẹo:

  • Đặt tên file là index.html để làm trang chính
  • Tạo thêm file style.css để tách riêng CSS
  • Tạo file script.js cho JavaScript

🔧 Cấu trúc file

Tổ chức file theo cấu trúc sau:

my-portfolio/ ├── index.html # Trang chính ├── style.css # File CSS chính ├── script.js # File JavaScript ├── images/ # Thư mục ảnh │ ├── profile.jpg # Ảnh cá nhân │ └── projects/ # Ảnh dự án └── assets/ # Tài nguyên khác

✅ Checklist Bước 2:

3

Tùy chỉnh CSS Layout

Thiết kế layout và màu sắc theo ý thích

🎨 Tùy chỉnh màu sắc

Thay đổi CSS variables để đổi màu toàn bộ website:

:root { /* Thay đổi màu chính */ --primary: #667eea; /* Màu chính */ --secondary: #764ba2; /* Màu phụ */ --accent: #10d5c2; /* Màu nhấn */ --text-dark: #2d3436; /* Chữ đậm */ --text-light: #636e72; /* Chữ nhạt */ }

📐 Điều chỉnh Grid Layout

Tùy chỉnh layout grid cho các section:

/* Skills Grid - Thay đổi số cột */ .skills-grid { display: grid; grid-template-columns: repeat(3, 1fr); /* 3 cột thay vì auto-fit */ gap: 2rem; } /* Portfolio Grid - Layout khác */ .portfolio-grid { display: grid; grid-template-columns: 2fr 1fr 1fr; /* Cột đầu to hơn */ gap: 2rem; }

✨ Thêm animations

Tạo hiệu ứng chuyển động cho các element:

/* Hover effect cho cards */ .skill-card { transition: all 0.3s ease; } .skill-card:hover { transform: translateY(-10px) scale(1.02); box-shadow: 0 15px 40px rgba(0,0,0,0.2); } /* Animation khi scroll */ @keyframes slideInUp { from { opacity: 0; transform: translateY(50px); } to { opacity: 1; transform: translateY(0); } }

💡 Mẹo CSS:

  • Sử dụng CSS Grid cho layout chính, Flexbox cho alignment
  • Dùng CSS variables để dễ thay đổi màu sắc
  • Thêm transition cho mọi hover effect
  • Test responsive trên nhiều kích thước màn hình

✅ Checklist Bước 3:

4

Thêm nội dung Portfolio

Upload ảnh và thêm dự án thực tế

📸 Thêm ảnh cá nhân

Thay thế ảnh placeholder bằng ảnh thật:

<!-- Thay đổi src của ảnh --> <img src="images/profile.jpg" alt="Ảnh của tôi">

💡 Mẹo về ảnh:

  • Kích thước ảnh nên từ 400x400px trở lên
  • Format: JPG hoặc PNG, nén để giảm dung lượng
  • Ảnh nên có ánh sáng tốt và rõ nét

💼 Thêm dự án Portfolio

Cập nhật section portfolio với dự án thật:

<div class="portfolio-item"> <img src="images/projects/project1.jpg" alt="Website thương mại điện tử"> <div class="portfolio-info"> <h3>Website Bán Hàng</h3> <p>Website thương mại điện tử với React, Node.js</p> <a href="https://github.com/username/project" class="portfolio-link"> Xem trên GitHub → </a> </div> </div>

📞 Cập nhật thông tin liên hệ

Thay đổi email, phone và social links:

<div class="contact-info"> <h3>Thông tin liên hệ</h3> <p>📧 Email: your.real.email@gmail.com</p> <p>📱 Phone: +84 xxx xxx xxx</p> <p>💼 LinkedIn: /in/your-linkedin</p> <p>🐙 GitHub: /your-github</p> </div>

⚠️ Bảo mật:

Cân nhắc kỹ trước khi public thông tin cá nhân như số phone. Có thể chỉ để email và social links.

✅ Checklist Bước 4:

5

Deploy và chia sẻ

Đưa website lên internet để mọi người xem

🚀 Deploy với GitHub Pages

Cách miễn phí và dễ nhất để host website:

# 1. Tạo repository mới trên GitHub # 2. Upload code lên GitHub git init git add . git commit -m "Initial portfolio" git remote add origin https://github.com/username/portfolio.git git push -u origin main # 3. Enable GitHub Pages trong Settings # 4. Website sẽ có link: username.github.io/portfolio

🌐 Deploy với Netlify

Platform hosting miễn phí với nhiều tính năng:

💡 Ưu điểm Netlify:

  • Drag & drop để deploy
  • Custom domain miễn phí
  • Auto deploy khi update code
  • HTTPS tự động

🔧 Tối ưu trước deploy

Checklist để website chạy tốt nhất:

/* Minify CSS */ /* Compress images */ /* Test trên mobile */ /* Kiểm tra loading speed */ /* Validate HTML/CSS */

📈 Theo dõi và cải thiện

Sau khi deploy, bạn có thể:

  • Thêm Google Analytics để theo dõi traffic
  • Test speed với PageSpeed Insights
  • Thu thập feedback từ bạn bè
  • Cập nhật nội dung định kỳ
